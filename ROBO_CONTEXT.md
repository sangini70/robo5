# ROBO_CONTEXT.md

## 1. 프로젝트 정체성

robo5는 단순 블로그가 아니다.

robo5는:

- AI 시대 금융 탐험 플랫폼
- 금융 교양 미디어
- 세계 흐름 탐험 허브
- SEO 기반 콘텐츠 엔진
- 체류형 정보 플랫폼

을 목표로 한다.

핵심은:

“사용자가 반복 방문하며
흥미롭게 금융과 세계 흐름을 탐험하도록 만드는 것”

이다.

---

## 2. 절대 금지 방향

robo5는 아래 플랫폼이 아니다.

- 투자 권유 플랫폼
- 리드 생성 플랫폼
- 수익 인증 플랫폼
- 자극형 클릭 유도 플랫폼
- 공포 마케팅 플랫폼
- AI 대량생성 콘텐츠 플랫폼
- SaaS형 대시보드 플랫폼

특히:

- 종목 추천
- 매수/매도 유도
- 수익 인증
- 선동형 제목
- 과장형 예측
- 공포 기반 클릭 유도

를 금지한다.

---

## 3. 핵심 목적

robo5의 목적은:

- 금융 이해
- 흥미
- 탐험
- 세계 흐름 연결
- AI 시대 금융 교양
- 사용자 체류시간 증가
- 반복 방문
- 자연스러운 콘텐츠 탐색
- 애드센스 기반 수익

이다.

핵심은:

“행동 강요”
가 아니라

“흥미로운 탐험 경험”

이다.

---

## 4. 핵심 UX 철학

robo5 UX는:

- 인간적인 흐름
- 탐험 리듬
- cognitive load 감소
- story-like navigation
- scroll rhythm
- 자연스러운 연결

을 중심으로 설계한다.

금지:

- AI 느낌 강한 UI
- 과도한 SaaS 느낌
- 카드 반복 피로
- 정보 과밀
- 지나친 CTA
- 의미 없는 인터랙션
- 과도한 애니메이션

중요:

사용자는
“정보를 소비”
하는 것이 아니라

“탐험”
해야 한다.

---

## 5. 콘텐츠 철학

콘텐츠는:

- 쉬워야 한다
- 흥미로워야 한다
- 세계와 연결되어야 한다
- 현재 흐름과 연결되어야 한다
- 인간적인 문체를 유지해야 한다

주요 콘텐츠 방향:

- AI와 금융
- ETF
- 환율
- 경제 흐름
- 세계 뉴스
- 금융 심리
- 금융 기초
- 과거와 현재 연결
- 글로벌 변화
- 인간과 기술 변화

금지:

- AI 슬롭
- 자동생성 티 나는 글
- 정보 짜깁기
- 의미 없는 장문
- SEO만 위한 글
- 키워드 남용

---

## 6. SEO 철학

robo5는 WordPress 플러그인 기반 SEO가 아니라
“자체 SEO 엔진 구조”
를 지향한다.

핵심:

- metadata
- JSON-LD
- canonical
- sitemap
- robots
- internal linking
- flow 연결
- category/tag 구조
- related content
- 체류시간 기반 SEO

중요:

SEO는
“검색 유입”
만이 아니라

“탐험 흐름”
까지 포함한다.

---

## 7. 운영 철학

robo5는:

- phased evolution
- incremental improvement
- production-first
- browser verification
- observable evidence

원칙을 따른다.

금지:

- 대형 리팩토링
- 추측 기반 수정
- 흐름 무시한 기능 추가
- localhost만 보고 판단
- 실제 UX 확인 없는 수정

중요:

브라우저에서 실제 어떻게 느껴지는지가
최종 진실이다.

---

## 8. 디자인 철학

디자인 핵심:

- AI스럽지 않을 것
- 인간적인 온도감 유지
- 지나치게 현대 SaaS 같지 않을 것
- 읽고 싶어지는 분위기
- 탐험하고 싶어지는 흐름
- 정보보다 리듬 우선

robo5는:

“도구”
보다

“탐험형 미디어”

처럼 느껴져야 한다.

---

## 9. 기술 방향

현재 robo5는:

- Next.js App Router
- React
- TypeScript
- Tailwind CSS
- JSON 기반 콘텐츠 구조

를 기반으로 한다.

핵심 방향:

- 복잡한 CMS 지양
- 단순하고 검증 가능한 구조 유지
- JSON 중심 콘텐츠 흐름 유지
- 최소 수정 원칙 유지
- production 검증 우선

robo5는 앞으로:

- 금융 탐험 플랫폼
- AI 금융 교양 플랫폼

## 11. 최종 정의

robo5는:

“AI 시대 금융 탐험 플랫폼”

이다.

사용자는:

- 배우기 위해 오고
- 탐험하기 위해 머물고
- 흐름을 따라 이동하며
- 자연스럽게 반복 방문한다.

robo5의 핵심은:

“정보”
가 아니라

“탐험 흐름”
이다.
